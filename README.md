# SIGV Web - Fase 1

Sistema Integral de Gestión de Visas - AmCham Atlántico y Magdalena.

## Incluye

- Login simple.
- Dashboard básico.
- Nuevo caso con flujo real de cliente nuevo.
- Cálculo automático de tarifas.
- Gastos adicionales según tipo de solicitud.
- Checklist de documentos antes de agendar.
- Tabla de casos registrados con datos locales.
- Tabla de tarifas.

## Ejecutar

```bash
npm install
npm run dev
```

Usuario de prueba: `admin`  
Clave: `1234`

## Nota

Esta fase todavía no conecta Google Sheets. Los datos son locales de prueba para validar el flujo con Milena y el equipo.
