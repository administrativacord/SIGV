# SIGV Web - Fase 1

Sistema Integral de Gestión de Visas - versión web inicial.

## Incluye
- Login básico
- Dashboard
- Clientes
- Casos
- Seguimiento
- Pagos
- Datos de prueba locales

## Usuario de prueba
- Usuario: `admin`
- Clave: `1234`

## Ejecutar localmente
```bash
npm install
npm run dev
```

## Compilar para Vercel
```bash
npm run build
```

## Nota
Esta fase no tiene conexión a Google Sheets todavía. Esa conexión se hará en una fase posterior.
